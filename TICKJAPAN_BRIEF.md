# 틱재팬(TickJapan) 서비스 브리핑

> 이 문서는 틱재팬에 대한 이해도가 높은 AI 에이전트를 위한 컨텍스트 파일입니다.
> 콘텐츠 전략, 투자 유치, 비즈니스 상담 등에 활용하세요.

---

## 1. 서비스 개요

**틱재팬(TickJapan)** 은 한국어 → 일본어 번역기 + 일본어 피치 악센트 학습 웹앱입니다.

- **URL**: Vercel 배포 (SPA)
- **타깃 유저**: 일본어를 독학하는 한국인, 특히 "문법·발음까지 제대로 배우고 싶은" 중급 학습자
- **핵심 차별점**: 단순 번역이 아닌 **발음(피치 악센트) + 문법 원리 해설**을 한 화면에서 제공

### 서비스가 풀려는 문제
- 파파고·구글 번역은 번역문만 주고 발음·악센트 정보는 없음
- 일본어는 같은 단어라도 고저(피치 악센트)가 틀리면 의미가 달라지는데, 기존 앱은 이를 다루지 않음
- 단어장 앱은 빈도순 정렬이나 활용형(활용표) 제공이 부실함

## 2. 핵심 기능

### ① 한→일 번역기 (메인 기능)
- 한국어 입력 → 일본어 번역 + 히라가나 독음 + 피치 악센트 그래프 **동시 제공**
- 번역 결과를 **문장 분해(Breakdown)** 형태로 의미 덩어리 단위로 해설
  - 예: "日本語を話す" → [日本語を(명사+조사)] + [話す(동사, 5단 활용)] + [ことができます(문법 패턴)]
  - 각 덩어리마다 히라가나·한국어 발음·품사·활용 원리 표시
- 비로그인 시 번역 5회 제한 → 로그인 유도

### ② 피치 악센트 그래프
- 일본어 특유의 고저 악센트를 시각적 그래프로 표시
- OJAD(일본어 악센트 DB) 연동 + Google Gemini 로컬 계산 병행
- TTS(Text-to-Speech)로 실제 발음 청취 가능 (여성/남성 선택)

### ③ 단어 라이브러리 (5개 품사)
BCCWJ(일본어 균형 코퍼스) 빈도순 정렬 기반 정적 데이터:

| 섹션 | 단어 수 | 내용 |
|------|---------|------|
| 동사 TOP100 | 100개 | する·いる·ある 등, 예문 3개씩 (총 300개 예문) |
| い형용사 TOP100 | 100개 | 多い·高い·大きい 등, 예문 3개씩 |
| な형용사 TOP100 | 100개 | 可能·好き·重要 등, 예문 3개씩 |
| 명사 TOP100 | 100개 | こと·人·自分 등, 예문 3개씩 |
| 조사 TOP10 | 10개 | は·が·を·に 등 핵심 조사 |

각 단어 상세 페이지에서 제공하는 것:
- **전체 활용표** (현재/과거/긍정/부정/의문/정중체/보통체)
- **예문 3개** + 각 예문마다 피치 그래프 + TTS
- **문법 패턴 해설** (예: 〜ことができる, 〜んだって 등)
- **활용 원리 보기**: AI(Gemini)가 문장을 의미 덩어리로 분해해 설명
- **인스타그램 무료 강의 배너** (품사별 연결)

### ④ 저장 기능
- **단어 저장**: 단어 카드에서 북마크 → localStorage 저장 (비로그인도 가능)
- **예문 저장**: 각 예문 옆 "저장하기" 버튼 → localStorage
- **번역 저장**: 번역 결과 저장 → 서버 DB (로그인 필요)
- **저장 목록 드로어**: 헤더의 "저장 목록" 버튼으로 오픈
  - 번역 저장 / 저장 단어 / 저장 예문 3탭
  - 저장 예문 클릭 시 해당 단어 상세 페이지 + 예문 섹션으로 자동 스크롤

### ⑤ 회원 시스템
- 회원가입: 이름 + 전화번호 (간편 가입, 비밀번호 없음)
- 로그인 상태에서 번역 저장 기록이 서버에 영구 보존
- 비로그인도 단어/예문 저장 가능 (localStorage)

## 3. 기술 스택

| 레이어 | 기술 |
|--------|------|
| 프론트엔드 | React 18 + Vite SPA, React Router DOM v7 |
| 백엔드 | FastAPI (Python), 단일 main.py |
| DB | PostgreSQL (SQLAlchemy ORM) |
| AI | Google Gemini gemini-2.5-flash-lite (번역·문장분해) |
| TTS | Google Cloud TTS (Wavenet 음성) |
| 발음 데이터 | OJAD (大阪大 일본어 악센트 DB) 연동 |
| 배포 | Vercel (프론트), Railway (백엔드) |

## 5. 데이터 규모

- 총 데이터 라인 수: 약 28,000줄 (정적 JS 데이터)
- 동사 300개 예문 패턴 + 각 형용사·명사도 동급 → **총 예문 패턴 약 1,200개**
- 각 예문마다: 일본어 원문, 한국어 번역, 히라가나 독음, 피치 악센트 데이터, 문법 패턴(이름·의미·설명) 포함

## 6. 비즈니스 모델 현황 및 방향

### 현재 수익화 구조
- **현재는 완전 무료** (광고·구독·인앱결제 없음)
- 번역 5회 제한으로 로그인 유도 → 유저 데이터 수집

### 잠재 수익화 방향 (아직 미결정)
- **프리미엄 구독**: 번역 무제한, 단어 저장 클라우드 동기화, 고급 학습 기능
- **B2B**: 일본어 학원·학교에 라이선스 제공
- **인스타그램 강의 연계**: 콘텐츠 유료화 (클래스101, 크몽 등)
- **광고**: 일본어 학습 관련 광고 게재

## 8. 서비스의 강점 요약

1. **학습 깊이**: 번역 + 발음 + 문법 원리를 한 화면에서 제공하는 앱은 드묾
2. **데이터 품질**: BCCWJ 실사용 빈도 기반으로 "실제로 자주 쓰는" 단어만 선별
3. **무료 진입 장벽 없음**: 비로그인도 핵심 기능 대부분 이용 가능 → 퍼널 상단 유입 용이
4. **콘텐츠 연계**: 인스타그램 무료 강의와 연동, SNS 트래픽 유입 구조 보유
5. **언론 노출**: 동아일보 IT 섹션 기사화 이력 (신뢰도 확보)
6. **개인 개발**: 창업자 1인이 프론트/백엔드/AI/데이터 전체 구축 → 빠른 의사결정·실행력

## 9. 서비스의 약점 및 도전 과제

1. **수익화 미정**: 현재 완전 무료로 운영 중, 유료 전환 시 이탈 리스크 존재
2. **AI 비용**: 번역·문장 분해에 Gemini API 호출 → 트래픽 증가 시 비용 부담
3. **경쟁 구도**: 파파고, 구글 번역, Duolingo, 시원스쿨 등 대형 경쟁자 존재
4. **모바일 앱 없음**: 현재 PWA(웹앱) 형태, 네이티브 앱 미출시
5. **콘텐츠 업데이트 한계**: 단어 데이터가 정적 JS 파일로 관리되어 콘텐츠 추가가 수동
